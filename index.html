    Como Jogar?

    Este é o seu robô amigo. Ele quer chegar na moeda, mas ele vai precisar da sua ajuda!

    Arraste os blocos coloridos para a caixa dizendo "Arraste para cá".

    Esses bloquinhos serão usados pelo robôzinho para se movimentar pelo mapa!

    Existem quatro tipos de blocos:

    • O azul é o bloco de se mover.
    • Os verdes são blocos de girar.
    • O laranja é o bloco de repetir

    Alguns níveis não irão apresentar todos os blocos, apenas os que você precisará usar naquele nível.

    Conforme for passando as fases, mais blocos irão aparecendo.

    O bloco laranja de repetir faz uma lista de blocos se repetirem mais de uma vez, quantas vezes for necessário!

    Para escolher quantas vezes repetir, coloque o número dentro da caixa cinza no bloco:

    O painel apresenta também, quatro botões:

    • O primeiro botão faz com que o robô execute os blocos que você arrastou para a caixa.
    • O segundo botão faz com que o robô pare de executar os blocos. Use ele caso tenha algum bloco errado na caixa.
    • O terceiro botão apaga todos os blocos da caixa. Cuidado com ele!
    • O quarto botão abre essa caixa de ajuda!

    Use a barra branca para mudar a velocidade do robôzinho.
